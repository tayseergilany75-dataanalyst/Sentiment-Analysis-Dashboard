# Sentiment-Analysis-Dashboard
This project analyzes social media data to understand user sentiment, engagement, and behavior patterns using Power BI.
Objectives:
Analyze sentiment distribution
Measure engagement (likes & retweets)
Identify top-performing content
Provide actionable recommendations
🛠 Tools Used:
Power BI
Excel
DAX
📊 Dashboard Features:
KPI Cards
Sentiment Analysis
Engagement Analysis
Time Analysis
Country Analysis
Dynamic Recommendations
💡 Key Insights:
Positive sentiment dominates user reactions
Engagement varies by platform
Peak posting time improves performance
🚀 Recommendations:
Focus on high-performing content
Optimize posting time
Improve engagement strategies
